# About Me

Hi, I'm Trevor.

I'm an undergraduate student double majoring in **Computer Science** and **Cybersecurity**. I'm interested in using technology safely, securely, and responsibly to solve real-world problems and build tools that can make a positive impact.

I enjoy working on data-driven and software-focused projects, including personal tools, developer utilities, cybersecurity projects, and larger systems that solve real problems or make new things possible through code and technology.

Currently, I'm learning more about **AI, LLMs, and ML** while continuing to improve my programming, cybersecurity, and problem-solving skills.

I'm especially interested in projects involving:

- Artificial intelligence and LLM-powered tools
- Cybersecurity, ethical hacking, and secure software
- Practical software development and problem-solving tools
- Machine learning, automation, and developer utilities

I'm always working to become a better developer by building projects, experimenting with new technologies, and learning from real-world problems.
